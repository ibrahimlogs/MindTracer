# Demo Script

## 120-second version

0–12s: Open `/demo/judge`, click “Start the two-minute demo,” and show the advertising/sales table. Narration: “Two learners both answer 10.”

12–28s: Show Learner A. Narration: “Learner A notices upward movement but does not use the repeated change.”

28–42s: Show Learner A verification. Narration: “MindTrace asks for the change from 1 to 2.”

42–55s: Show Learner A intervention. Narration: “The smallest useful support is the +2 repeated difference.”

55–70s: Show Learner B. Narration: “Learner B uses a direct multiplication rule.”

70–84s: Show Learner B verification. Narration: “The system checks whether double advertising matches the table.”

84–98s: Show Learner B intervention. Narration: “This learner needs the offset conflict, not the same cue as Learner A.”

98–112s: Show transfer. Narration: “A new context checks independent use.”

112–120s: Show Reasoning Delta and report link. Narration: “MindTrace measures changed reasoning, not just the corrected answer.”

## 180-second backup

Use the same sequence with slower pauses on the architecture and evaluation summaries.

## Cursor and recording checklist

- Record at 1080p or higher.
- Hide personal browser data and notifications.
- Use consistent zoom and a readable cursor.
- Check audio before recording.
- Restart Judge Mode before recording.
- Confirm cached fallback is visible.
- Do not wait through long loading states.
- Show only one architecture screen.
- End with project name and repository.
- Do not use copyrighted music unless licensed.
