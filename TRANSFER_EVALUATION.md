# Transfer evaluation

Command:

```bash
corepack pnpm evaluate:transfer -- --mode=deterministic
```

Label: Prototype transfer evaluation.

The harness checks answer-correctness agreement, concept-applied agreement, transfer-status agreement, independence-state agreement, guess detection, arithmetic-vs-concept distinction, unsupported-success rate, prohibited-output rate, fallback rate, and local latency.

This is a prototype evaluation harness, not a benchmark of learning effectiveness.
