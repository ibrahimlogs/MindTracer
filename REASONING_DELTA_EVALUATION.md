# Reasoning Delta evaluation

Command:

```bash
corepack pnpm evaluate:reasoning-delta -- --mode=deterministic
```

Label: Prototype Reasoning Delta evaluation.

The harness checks dimension/change agreement, overall-change agreement, preserved-understanding agreement, remaining-gap agreement, transfer-readiness agreement, unsupported-improvement rate, prohibited-output rate, fallback rate, and local latency.

This is a prototype safety and regression harness, not an educational outcome claim.
