# Known Limitations

- The educational dataset is intentionally small and curated.
- Prototype evaluations are regression and safety checks, not broad efficacy studies.
- Live OpenAI verification is pending unless credentials are configured and smoke tests are run.
- Live PostgreSQL verification is pending unless a safe test database is configured.
- Judge Mode can use reviewed cached responses; it never labels cached output as live AI.
- No authentication, payments, teacher dashboards, voice input, diagram recognition, or free-form tutoring chat are implemented.
- Deployment and demo video recording remain external submission tasks.
